💾 Respaldo y restauración

Guarda un respaldo completo de stock, historial e instalaciones. No incluye las fotos (esas se guardan aparte en el dispositivo para no llenar el respaldo). Si borras algo por error, puedes restaurarlo desde el archivo.

⚠ Zona peligrosa

Borra todos los datos de la aplicación y vuelve al estado inicial. Útil para comenzar desde cero. Esta acción no se puede deshacer.

📷 Respaldo de fotos (opcional)

Las fotos viven en este dispositivo y navegador. Si vas a cambiar de celular o borrar el navegador, exporta un respaldo de fotos para no perderlas. Este archivo puede ser pesado según cuántas fotos tengas.
